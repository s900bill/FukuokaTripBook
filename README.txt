福岡旅遊小幫手 (Fukuoka Trip Book)
================================

一個為福岡旅遊設計的輕量級 PWA 工具，提供行程管理、匯率換算、優惠券展示與常用日語翻譯功能。

1) 直接開啟方式
--------------
建議使用簡單的 Local Server 以獲得最佳體驗：
- VSCode Live Server
- 或使用 Python: `python -m http.server 8000`

2) 主要功能
----------
- **行程規劃**：讀取 trip.json 顯示每日行程與景點地圖。
- **匯率換算**：支援台幣/日幣即時匯率轉換。
- **優惠券**：內建常用商店優惠券，已優化為 WebP 格式以節省流量。
- **日語翻譯**：收錄旅遊常用句型，支援語音播放（TTS），包含長輩友善與我是台灣人等短語。

3) 技術細節
----------
- **PWA**：支援離線使用（透過 sw.js 緩存）。
- **圖片優化**：優惠券圖片採用 WebP 格式，平均體積縮減約 80%。
- **資料驅動**：所有行程資料皆由 trip.json 定義，方便自定義。

4) 離線使用
----------
本工具支援 PWA 離線模式。在飯店連接 Wi-Fi 時開啟一次，Service Worker 就會自動快取 App Shell 與圖片，確保在外出沒網路時仍能展示優惠券。
